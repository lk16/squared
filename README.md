Squared
=======

A graphical othello/reversi program with (strong) AI/bot/robot.

Designed to run on *nix 64-bit systems.

Dependencies
------------
* gtkmm-dev
* glibmm-dev
* libboost-dev
* sqlite3-dev


Local installation
-----------
* install dependencies
* ``cd squared``
* ``git clone https://github.com/lk16/squared
* ``git submodule init``
* ``git submodule update``
* ``cmake .``
* ``make -j \`nproc\```
